# AI Ecologist

Structural ecological intelligence engine. The living network, and where it breaks first.

An interactive scrollytelling site that turns a species inventory into a map of what
holds an ecosystem together, and where it fails first. It is anchored by the Oriente
cloud-forest case study and a cross-biome structural reference library, with a linked
product brief.

## Run it

The page is fully self-contained and runs offline. It loads its data files, so serve it
over http rather than opening from the file system:

```
python3 -m http.server 8000
# then open http://127.0.0.1:8000/
```

## Honesty

Structural reconstruction and fragility are benchmarked and reproducible. Financial
outputs are prototype, screening-grade, and not market-calibrated, and the site shows no
specific economic figures. Cross-biome similarity is structural, not taxonomic.
